# Decisions

This directory contains the decision records for esmf-vs-code-plugin.

For new ADRs, please use [adr-template.md](adr-template.md) as the basis. More
information about MADR is available at <https://adr.github.io/madr/>. General
information about architectural decision records (ADRs) is available at
<https://adr.github.io/>.
